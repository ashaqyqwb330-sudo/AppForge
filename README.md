# AppForge - صانع التطبيقات الذكي

AppForge هو تطبيق أندرويد أصلي يحول أي ملف قاعدة بيانات SQLite إلى تطبيق حقيقي بواجهات متعددة ومتنوعة.
ما عليك سوى اختيار ملف .db، ويقوم التطبيق بتحليله تلقائياً واقتراح قالب مميز لعرض محتواه، مع إمكانية إدارة عدة قواعد بيانات والتبديل بينها.

## المميزات
- دعم أي قاعدة بيانات SQLite
- أربعة قوالب عرض افتراضية (معرض، جدول، قصص، شبكة)
- واجهة عربية كاملة مع دعم RTL
- نظام Material 3 مع ألوان ديناميكية
- عرض ذكي للبيانات (روابط، إيميلات، صور، تواريخ)
- إدارة متعددة للتطبيقات الفرعية

## المتطلبات
- Android Studio Flamingo أو أحدث
- JDK 17
- Android SDK 34

## التشغيل
1. افتح المشروع في Android Studio.
2. انتظر مزامنة Gradle.
3. شغل التطبيق على محاكي أو جهاز حقيقي.

## بناء APK
```bash
./gradlew assembleDebug   # لنسخة التطوير
./gradlew assembleRelease # لنسخة الإصدار (يتطلب توقيع)
```
